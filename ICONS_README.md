## Design Guidelines

- Use a simple, recognizable icon related to music or ear training
- Suggestions: musical note, ear symbol, treble clef, or abstract sound waves
- Use the app's color scheme:
  - Background: `#FAF7F2` (Soft Oatmeal)
  - Accent: `#E6D9F6` (Gentle Lavender)
  - Text: `#5A5651` (Soft Charcoal)
- Make sure the icon works well on both light and dark backgrounds
- Keep it simple - it needs to be recognizable at small sizes

## Tools for Creating Icons

- **Figma** - Free design tool with export to PNG
- **Canva** - Easy online design tool
- **GIMP** - Free image editor
- **Inkscape** - Free vector graphics editor
